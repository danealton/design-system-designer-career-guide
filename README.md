# Design System Designer Career Guide

> Комплексное руководство для продуктовых дизайнеров, специализирующихся на разработке дизайн-систем и UI-kit'ов

## О проекте

Этот репозиторий содержит материалы для подготовки к карьере продуктового дизайнера UI-kit'ов:

- Framework компетенций по уровням
- Вопросы для интервью и теоретические гайды
- Примеры резюме
- Глоссарий терминов

## Структура репозитория

```text
├── competency-framework/     # Описание компетенций по уровням
├── interview-prep/          # Материалы для подготовки к интервью
│   ├── interview-questions/ # Вопросы для интервью
│   └── theoretical-guides/  # Теоретические гайды
├── cv-examples/             # Примеры резюме для разных уровней
├── GLOSSARY.md              # Глоссарий терминов
└── README.md
```

## Разделы

### 1. [Framework компетенций](competency-framework/)

Детальное описание требований к продуктовому дизайнеру UI-kit'ов на разных уровнях: от Junior до Lead. Начните отсюда, чтобы понять, какие навыки и знания требуются на каждом уровне.

### 2. [Подготовка к интервью](interview-prep/)

Материалы для подготовки к техническим и продуктовым интервью:

- [Вопросы для интервью](interview-prep/interview-questions/) - структурированные вопросы по уровням с ключевыми моментами для ответов
- [Теоретические гайды](interview-prep/theoretical-guides/) - детальные разборы ключевых тем для глубокой подготовки

### 3. [Примеры резюме](cv-examples/)

Примеры структуры и содержания резюме для различных уровней позиций. Помогут понять, как презентовать опыт и навыки.

### 4. [Глоссарий](GLOSSARY.md)

Справочник ключевых терминов с краткими определениями и ссылками на детальные теоретические гайды. Используйте для быстрого поиска определений.

## Как использовать

1. **Для дизайнеров:** Используйте материалы для подготовки к интервью и планирования карьерного роста
2. **Для рекрутеров:** Используйте framework компетенций для оценки кандидатов
3. **Для менеджеров:** Используйте для планирования развития команды
