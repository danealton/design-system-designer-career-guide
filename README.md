# Design System Designer Career Guide

> Комплексное руководство для продуктовых дизайнеров, специализирующихся на разработке дизайн-систем и UI-kit'ов

## О проекте

Этот репозиторий содержит материалы для подготовки к карьере продуктового дизайнера UI-kit'ов:

- Framework компетенций по уровням
- Примеры резюме
- Материалы для подготовки к интервью
- Теоретические гайды по ключевым темам

## Структура репозитория

```text
├── competency-framework/     # Описание компетенций по уровням
├── cv-examples/             # Примеры резюме для разных уровней
├── interview-prep/          # Материалы для подготовки к интервью
│   ├── theoretical-guides/  # Теоретические гайды
│   └── interview-questions/ # Вопросы для интервью
└── README.md
```

## Разделы

### [Framework компетенций](competency-framework/)

Детальное описание требований к продуктовому дизайнеру UI-kit'ов на разных уровнях: от Junior до Lead.

### [Примеры резюме](cv-examples/)

Примеры структуры и содержания резюме для различных уровней позиций.

### [Подготовка к интервью](interview-prep/)

Материалы для подготовки к техническим и продуктовым интервью.

## Как использовать

1. **Для дизайнеров:** Используйте материалы для подготовки к интервью и планирования карьерного роста
2. **Для рекрутеров:** Используйте framework компетенций для оценки кандидатов
3. **Для менеджеров:** Используйте для планирования развития команды
